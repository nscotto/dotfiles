Manual installation
===================

FZF
---

### Ubuntu

```bash
sudo apt-get install fzf
```

Lazygit
-------

### Ubuntu

```bash
sudo add-apt-repository ppa:lazygit-team/release
sudo apt-get update
sudo apt-get install lazygit
```
